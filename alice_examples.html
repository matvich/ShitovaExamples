Галерея аудио-примеров

Примеры сгруппированы по аспектам и значениям: эмоции, скорость, тон, громкость и любые другие категории.
Как подключить свои аудио

Все данные вшиты прямо в HTML. Описания категорий лежат в descriptions, аудио — в audioData.

const audioData = {
  Emotions: {
    Sad: [
      {
        url: "Tags/Emotions/Sad/sad_001.wav",
        text: "Грустная интонация, замедленная речь, мягкое затухание фразы."
      }
    ]
  }
};
